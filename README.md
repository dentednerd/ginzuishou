# ginzuishou

Ginzuishou is a visual theme stylesheet for my Codepen projects, built in Sass.

## Usage

Add the stylesheet in the `<head>` section of your HTML:

```html
<link rel="stylesheet" href="https://dentednerd.github.io/ginzuishou/index.css" />
```

## Features

- styles for many (but not all) HTML elements
- a set of CSS classes for quick visual theming
- light and dark modes
- a body background resembling a blueprint grid
- CSS custom properties, based on Sass variables

## Development

`npm run dev` runs Sass in `--watch` mode.
